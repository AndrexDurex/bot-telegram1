# BioAgent (Personal Health Assistant)

BioAgent is a specialized AI assistant that helps manage personal routines, health habits (based on Dr. La Rosa's protocols), schedule (Google Calendar), and tasks.

## Features
- Context-Aware Memory (Firebase RTDB)
- Schedule Integration (Google Calendar)
- Task Management
- RAG Health Database

## Deployment (Hugging Face Spaces)
This project is configured as a Docker Space on Hugging Face.
